# Support

This repository is shared as-is, with no paid support or service-level agreement.

## Where to ask for help

Use GitHub issues for:

- Broken install instructions.
- Broken links or missing files.
- Reproducible script errors.
- Clear suggestions for improving a skill.

Before opening an issue, please check the skill's `SKILL.md` and any setup notes inside that skill folder.

## What not to share

Do not post private documents, client files, legal matters, API keys, passwords, screenshots with personal data, or logs containing secrets.

If your issue depends on private material, replace it with a small synthetic example that shows the same problem.
