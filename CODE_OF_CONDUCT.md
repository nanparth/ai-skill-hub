# Code of Conduct

## Our standard

This project should be a practical and respectful place to share AI skill workflows.

Examples of welcome behavior:

- Being clear, patient, and constructive.
- Respecting different experience levels.
- Giving actionable feedback.
- Pointing out risks without personal attacks.

Examples of unacceptable behavior:

- Harassment, insults, threats, or personal attacks.
- Sexualized language or imagery.
- Publishing someone else's private information.
- Trolling, repeated disruption, or bad-faith participation.
- Encouraging unsafe use of credentials, private data, or confidential material.

## Scope

This code of conduct applies in repository issues, pull requests, discussions, and any project spaces connected to this repository.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or block participants who violate this code of conduct.

If you need to report a conduct issue, contact the maintainer through the repository issue tracker, GitHub profile, or any private contact method listed for the project. Do not include sensitive personal details in a public issue.
