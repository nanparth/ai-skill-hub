# README Self-Review

Check the README before presenting it.

## Reader Fit

- [ ] The first paragraph says what this is and who it helps.
- [ ] A beginner can find the install or quick-start path within 30 seconds.
- [ ] Required and optional dependencies are clearly separated.
- [ ] Examples use neutral placeholder names and paths.

## Source Fidelity

- [ ] Every command, script path, and option was checked against source files.
- [ ] The README does not describe files or features that are not present.
- [ ] Known limitations are stated plainly.

## Public Safety

- [ ] No private names, private project names, secrets, or local machine paths.
- [ ] No private workspace metadata or hidden workspace-only fields.
- [ ] No required dependency on a separate private document app, document-conversion tool, or private automation tool.
- [ ] Links and local file references are valid.
